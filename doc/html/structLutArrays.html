Detailed Description

LUT output arrays — shape [n_aod × n_h₂o × n_wl] (C row-major).

Element access: idx = iaod*n_h2o*n_wl + ih2o*n_wl + iwl

All mandatory arrays (R_atm … s_alb) must be pre-allocated by the caller. Optional pointers (T_down_dir, R_atmQ, R_atmU) are populated only when non-NULL on input; pass NULL to skip.
