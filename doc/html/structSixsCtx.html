Detailed Description

Master 6SV computation context — one instance per OpenMP thread.

Aggregates all Fortran COMMON-block equivalents. The Gauss quadrature arrays rm and gb are stored with an offset of MU_P so that the original negative indices (−MU_P … MU_P) map to [0 … 2·MU_P]. Use the convenience macros RM() and GB() for indexed access.
