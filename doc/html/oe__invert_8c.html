Joint per-pixel AOD + H₂O retrieval by grid-search MAP inversion.

For each pixel, tests every (AOD_i, H₂O_j) LUT grid point, selects the minimum-cost point, and refines with parabolic interpolation. Pixels with all-NaN VIS measurements are filled with the prior values.

Computation is OpenMP-parallelised over pixels; each thread uses a stack-allocated 64-band work buffer.

Parameters
[in]cfgLUT grid specification (wavelengths, AOD/H₂O grid, geometry).
[in]lutPre-computed LUT arrays.
[in]rho_toa_visTOA reflectance at VIS diagnostic bands, band-interleaved: [npix × n_vis]. Pixel i at band b: rho_toa_vis[i*n_vis + b].
[in]npixNumber of pixels.
[in]n_visNumber of VIS diagnostic bands (e.g. 4 for 470/550/660/870 nm).
[in]vis_wlCentral wavelengths of the VIS bands [µm], length n_vis.

[in]L_940TOA radiance near 940 nm [npix]; NULL → disable J_H₂O.
[in]L_1040TOA radiance near 1040 nm [npix]; NULL → disable J_H₂O.
[in]sza_degSolar zenith angle [degrees].
[in]vza_degView zenith angle [degrees].
[in]aod_priorScene-mean AOD prior (e.g. from retrieve_aod_ddv()).
[in]h2o_priorScene-mean H₂O prior [g/cm²] (e.g. from retrieve_h2o_940()).
[in]sigma_aodPrior uncertainty in log-AOD space (typical: 0.5).
[in]sigma_h2oPrior uncertainty in H₂O [g/cm²] (typical: 1.0).
[in]sigma_specObservation σ for spectral smoothness cost [reflectance units] (typical: 0.01).
[out]out_aodPre-allocated float[npix]; retrieved AOD at 550 nm.
[out]out_h2oPre-allocated float[npix]; retrieved WVC [g/cm²].
