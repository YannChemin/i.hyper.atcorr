Bilinear LUT interpolation at a fixed (AOD, H₂O) point.

Fills per-wavelength correction arrays [n_wl] by bilinear interpolation of the 3-D LUT along the AOD and H₂O axes. Useful for pre-computing scene-average atmospheric parameters before the pixel-level inversion loop.

Parameters
[in]cfgLUT grid specification.
[in]lutPre-computed LUT arrays.
[in]aod_valAOD at 550 nm (clamped to LUT range).
[in]h2o_valWater-vapour column [g/cm²] (clamped to LUT range).
[out]RsPath reflectance [n_wl].
[out]TdsDownward transmittance [n_wl].
[out]TusUpward transmittance [n_wl].
[out]ssSpherical albedo [n_wl].
[out]TddsDirect downward transmittance [n_wl]; pass NULL to skip (requires lut->T_down_dir to be non-NULL).
