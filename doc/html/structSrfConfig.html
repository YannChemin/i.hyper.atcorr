Detailed Description

Configuration for Gaussian SRF gas-transmittance correction.

Drives atcorr_srf_compute() to replace the coarse 6SV Curtis-Godson gas parameterisation with libRadtran reptran fine-resolution (~0.05 nm) gas transmittance convolved with the sensor spectral response function.
