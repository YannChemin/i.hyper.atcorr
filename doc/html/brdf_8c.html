Detailed Description

BRDF surface models — C port of 6SV2.1 BRDF Fortran subroutines.

Each kernel function provides a single-point evaluation; the Fortran originals loop over (μ, φ) Gauss grids but the inner formula is geometry-only, so we extract it for single (SZA, VZA, RAA) calls.

Models implemented (single-point): LAMBERTIAN — constant (handled by caller; brdf_eval returns 0) RAHMAN — RAHMBRDF.f (RPV) ROUJEAN — ROUJBRDF.f HAPKE — HAPKBRDF.f OCEAN — simplified Cox-Munk glint + whitecaps + water body WALTHALL — WALTBRDF.f MINNAERT — MINNBRDF.f ROSSLIMAIGNAN — ROSSLIMAIGNANBRDF.f (rlmaignanbrdf) VERSFELD — stub (mvbp1 external dependency not ported) IAPI — stub (complex canopy model not ported)

Bihemispherical (white-sky) albedo by numerical integration.

Integrates sixs_brdf_eval() over the outgoing hemisphere using a midpoint quadrature grid of size n_phi × n_theta.

Used by the BRDF-coupled inversion formula in atcorr_invert_brdf().

Parameters
[in]typeBRDF model selector.
[in]paramsModel parameters.
[in]cos_szacos(solar zenith angle).
[in]n_phiAzimuth quadrature points (suggested: 48).
[in]n_thetaZenith quadrature points (suggested: 24).
Returns
White-sky albedo ρ̄ ∈ [0, 1].

Evaluate BRDF at a single geometry point.

Returns the bidirectional reflectance factor ρ_s (dimensionless) for the specified model and geometry. Returns 0.0 for BRDF_LAMBERTIAN (the Lambertian coupling is handled analytically by atcorr_invert()).

Parameters

[in]paramsModel-specific parameters (use the matching union member).
[in]cos_szacos(solar zenith angle).
[in]cos_vzacos(view zenith angle).
[in]raa_degRelative azimuth angle [degrees, 0 = forward scatter].
Returns
Directional reflectance ρ_s ≥ 0.
