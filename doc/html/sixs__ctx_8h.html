Detailed Description

Internal 6SV2.1 computation context — C replacement for Fortran COMMON blocks.

Each LUT computation thread owns one SixsCtx instance allocated on the heap. Separating per-thread context from global state enables OpenMP parallelism over the AOD grid without locks.

The struct layout mirrors the Fortran COMMON blocks in the 6SV2.1 source:

  • /sixs_atm/ → SixsAtm
  • /sixs_del/ → SixsDel
  • /sixs_aer/ → SixsAer
  • /sixs_disc/ → SixsDisc (DISCOM output at 20 reference wavelengths)
  • /sixs_polar/→ SixsPolar (Legendre coefficients from TRUNCA)
  • etc.
Warning
After modifying this header, always rebuild with make clean && make. The Makefile has no automatic header-dependency tracking; a stale SixsDisc layout causes a segfault inside sixs_gauss.
