Detailed Description

DISCOM output at NWL_DISC reference wavelengths (/sixs_disc/).

Populated by sixs_discom() for a single AOD point; subsequently interpolated to sensor wavelengths in sixs_interp() / sixs_interp_polar().

All arrays have dimension [3][NWL_DISC] for [Rayleigh, mixed, aerosol].
