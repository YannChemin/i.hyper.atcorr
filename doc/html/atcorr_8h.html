Detailed Description

Public API for the i.hyper.atcorr atmospheric correction library.

Declares the LUT grid specification (LutConfig), output arrays (LutArrays), and all functions exported by libatcorr.so:

  • atcorr_compute_lut() — build the 3-D [AOD × H₂O × λ] LUT via DISCOM
  • atcorr_lut_slice() — bilinear interpolation at a fixed (AOD, H₂O)
  • atcorr_lut_interp_pixel() — trilinear point lookup for per-pixel paths
  • atcorr_invert() — Lambertian BOA reflectance inversion
  • atcorr_invert_brdf() — BRDF-coupled inversion (white-sky albedo term)
  • sixs_E0() — Thuillier solar irradiance spectrum
  • sixs_earth_sun_dist2()— seasonal Earth–Sun distance
  • SRF gas-transmittance correction (SrfConfig, atcorr_srf_compute(), …)

int atcorr_compute_lut ( const LutConfig *  cfg,
LutArrays *  out 
)

Compute a full 3-D atmospheric correction LUT.

Runs DISCOM once per AOD grid point (OpenMP-parallel over AOD dimension). Gas transmittance is applied at each (H₂O, λ) combination via the reptran parameterisation inside the AOD loop.

Parameters
[in]cfgLUT grid specification and scene geometry.
[out]outPre-allocated output arrays (R_atm … s_alb must be n_aod×n_h2o×n_wl floats each; optional pointers T_down_dir, R_atmQ, R_atmU populated when non-NULL).
Returns
0 on success; negative error code on failure.

Trilinear LUT interpolation at a single (AOD, H₂O, λ) point.

Performs binary-search bracketing + 8-corner trilinear interpolation to return a single set of atmospheric correction parameters. All values are clamped to the LUT grid boundaries; no extrapolation.

Used for per-pixel correction with spatially varying AOD/H₂O maps and for AOD-perturbation uncertainty estimation in uncertainty_compute_band().

Parameters
[in]cfgLUT grid specification.
[in]lutPre-computed LUT arrays.
[in]aod_valAOD at 550 nm.
[in]h2o_valWVC [g/cm²].
[in]wl_umWavelength [µm].
[out]R_atmPath reflectance.
[out]T_downDownward transmittance.
[out]T_upUpward transmittance.
[out]s_albSpherical albedo.

Fills per-wavelength correction arrays [n_wl] by bilinear interpolation of the 3-D LUT along the AOD and H₂O axes. Useful for pre-computing scene-average atmospheric parameters before the pixel-level inversion loop.

Compute per-band, per-H₂O SRF gas-transmittance correction factors.

Runs 4 × n_h2o uvspec subprocesses (fine/coarse × down/up × each H₂O) parallelised with OpenMP. The uvspec binary is located via PATH, $LIBRADTRAN_DIR/bin, or $GISBASE/bin.

Parameters
[in]srf_cfgSRF configuration (FWHM array and threshold).
[in]lut_cfgLUT grid specification (wavelengths, H₂O grid, geometry).
Returns
Pointer to correction table, or NULL if uvspec is not found or no bands fall below the FWHM threshold.
