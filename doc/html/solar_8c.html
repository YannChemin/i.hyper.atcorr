◆ sixs_E0()

float sixs_E0 ( float  wl_um)

Thuillier solar irradiance spectrum.

Returns E₀ in W/(m² µm) at wavelength wl_um, linearly interpolated from the 6SV2.1 Thuillier (2003) reference spectrum. Clamped to zero outside [0.25, 4.0] µm.

Parameters
[in]wl_umWavelength [µm].
Returns
E₀ [W m⁻² µm⁻¹] ≥ 0.

double sixs_earth_sun_dist2 ( int  doy)

Squared Earth–Sun distance for a given day of year.

Uses the approximation (Liou 2002), so perihelion (DOY≈3) gives d²<1 and aphelion (DOY≈185) gives d²>1.

Parameters
[in]doyDay of year [1, 365].
Returns
d² [AU²].
